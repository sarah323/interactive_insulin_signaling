# Interactive Biomedical Learning

Interactive Biomedical Learning is an open educational initiative focused on developing interactive web-based tools that make complex concepts in biochemistry, molecular biology, and laboratory medicine more approachable through visualization and active exploration.

This repository hosts standalone educational modules developed using HTML, CSS, and JavaScript. Each module is designed to complement—not replace—traditional lectures, textbooks, and practical teaching.

---

# Current Module

## Insulin Signaling Explorer

An interactive educational simulator that visualizes insulin signaling and insulin resistance through dynamic pathway exploration.

The simulator enables learners to:

- Explore the insulin signaling cascade step-by-step.
- Compare normal insulin signaling with insulin-resistant states.
- Understand the roles of major signaling molecules.
- Visualize metabolic and mitogenic signaling branches.
- Connect molecular mechanisms to metabolic disease.

> **Educational Note**
>
> This simulator uses simplified relative activity values to illustrate biological concepts. Numerical values are conceptual and should not be interpreted as experimentally measured biological quantities. The simulator is intended solely for educational purposes.

---

# Learning Objectives

After completing this activity, learners should be able to:

- Explain insulin receptor signaling.
- Describe the functions of IRS proteins, PI3K, AKT, mTOR, FOXO1, and GLUT4.
- Differentiate between metabolic and mitogenic signaling pathways.
- Explain the molecular basis of insulin resistance.
- Relate signaling abnormalities to metabolic disease.
